Connect to Cloud

To sync job cards across all staff phones and PCs in real-time, connect a free Firebase project. One-time setup — takes about 3 minutes.

  1. Open console.firebase.google.com and sign in
  2. Click Add project → name it brackenfell-gas → Continue
  3. In your project: click ⟨/⟩ Web → register app → copy the firebaseConfig
  4. In the left menu: Firestore Database → Create database → Start in test mode
  5. In the left menu: Storage → Get started → Start in test mode
  6. Paste your config below and click Connect
⚠ Invalid config — paste the full firebaseConfig object.
